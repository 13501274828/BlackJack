# BlackJack-21-point
This is an open-source 21-point game based on Java, aiming to simulate the real poker game experience. The game provides a simple and easy-to-use text interface, and players can interact with the game through the command line. 这是一个基于Java的开源21点游戏，希望模拟真实的扑克牌游戏体验。游戏提供简单易用的文本界面，玩家可以通过命令行与游戏进行交互。
